# Chpater 2. 컴퓨터 개요

#### 컴퓨터
+ 컴퓨터의 유래 : 주로 군사기술 분야에서, 미사일이나 포탄 등을 발사할 때, 각도/거리 등의 복잡한 계산을 빠르게 수행하기 위해 개발됨.
    
    - 데이터가 들어가면, 해당 데이터가 처리되어 그 결과를 보조기억 장치에 저장하거나, 표시(Display) 하거나 출력하여 결과를 나타내는 기능을 하는 기계
    - 스마트폰, 태블릿 PC도 다 컴퓨터에 속한다. 

+ 정보 : 컴퓨터에서 처리해 나온 의미 있는 자료가 정보가 된다.
+ 컴퓨터 안에서는 0과 1, On-Off만 인식을 할 수 있다.
+ 프로그램 : CPU가 일을 처리하기 위해, 명령어들의 집합을 적절히 모아 기능하도록 한 것.

+ 컴퓨터의 구성 : 
    - 하드웨어와 소프트웨어로 구성됨.
    - 하드웨어 : 컴퓨터에 보이는 물리적인 장치를 모두 하드웨어라고 부른다 
    - 소프트웨어 : 컴퓨터가 수행될 수 있도록, 명령어들의 집합(프로그램)으로 구성된 것을 모두 총칭해 소프트웨어라 부른다.

+ 보조 기억 장치에 데이터들이 저장될 때, "파일(File)"로 조직화되어 처리된다!

#### 하드웨어 
+ 입력 장치 : 데이터를 컴퓨터 내부로 입력하는 장치
    -  키보드, 마우스, 터치스크린, 카메라, 스캐너, 마이크, 바코드 등 데이터를 컴퓨터 상에 입력하는 것을 모두 입력장치라 부른다. 

+ 중앙처리장치(CPU : Central Processing Unit) : 
    - 제어 장치, 연산 장치, 레지스터로 구성된다.
    - CPU가 컴퓨터에 있는 모든 "작업"을 담당한다. 주기억 장치(메모리)에, 처리할 데이터와 프로그램을 올려두고 처리함.
    - 보통 컴퓨터에서 "마이크로 프로세서(보통 프로세서라고 부름) 라고 부르는 것들은 CPU 내의 부품을 의미한다.
    - 간단한 가전제품 같은 곳에도 이런 마이크로프로세서가 들어간다. 
    - 컴퓨터 안에 들어가는 CPU, 그 안에 일부분으로 들어가게 된다.

    - 제어 장치 : 제어 신호를 만들어 내고, 제어하는 장치
    - 연산 장치 : 컴퓨터 안에서 여러가지 산술연산, 논리연산을 수행하는 ALU
    - 레지스터 : CPU 안에서, 연산에 사용되는 자료 데이터들을 임시로 저장하는 곳.
    
    - CPU는 주로 메인보드 내의 전용 소켓에 장착이 된다. 
    - 크게 인텔 계열의 CPU와, AMD 계열의 CPU가 있다. 어떤 CPU를 선택하던, 그에 맞는 메인보드를 선택해야 한다.
    - **번외** : 
        - RAM(Random Access Memory) : 주기억장치에 사용되는 메모리. 길다랗게 메인모드에 꽂는다.
        - ROM(Read Only Memory) : 읽기만 할 수 있음. 쓰기는 할 수 없다. 주로 바이오스(BIOS) 프로그램이 내장되어 있다. 
        - BIOS(Basic Input Output System) : 컴퓨터 전원을 켰을 때, 컴퓨터의 주변 장치들을 점검해줌. 그래서 최종적으로 컴퓨터가 쓸 수 있는 상태인지 점검함.(Power On Self Test)
        - BIOS는 운영체제를 주기억 장치에 올리는 역할도 수행한다. 
  
+ 컴퓨터 시스템의 예시 :
    - 매장 내 키오스크, 편의점 계산대의 포스 시스템, 갤럭시 버즈(입력, 출력 모두 가능)

+ GPU(Graphic Processing Unit) : 그래픽 연산을 처리하기 위한 프로세서.
    - 그래픽은, 2차원 Matrix(행렬)로 볼 수 있다.
    - GPU는, 사이즈가 큰 Matrix(행렬)과 벡터를 다루는 복잡한 연산을 하는데 있어 CPU보다 뛰어난 성능을 가지고 있다.
    - 그래픽 연산 뿐 아니라, CPU 기능을 대신해 복잡한 연산(코인 채굴 등)에도 GPU가 사용되기도 한다.  
    - 범용 GPU(General Purpose GPU) : 단순한 그래픽 연산 뿐 아니라, CPU 기능을 대신에 복잡한 연산을 하는데 사용되는 GPU. 인공지능 분야의 Deep Neural Network에도 GPU를 사용하기도 한다.

+ 모바일 AP(Application Processor) : 스마트폰, 태블릿 PC 내에 들어가는 CPU.
    - AP 내에는 기본적인 CPU 기능 외에도, 영상 처리하는 GPU, 카메라 저장장치, 무선통신 인터페이스 제어, DSP(Digital Signal Processing) 등 각종 처리기능을 담당하고 있음.
    - System On Chip 형태를 취하고 있다. 하나의 칩 안에 여러 개의 반도체들이 들어가져 있음.
    - 예시 
        - 스냅드래곤(퀄컴)
        - 엑시노스(삼성)
        - 테그라(NVIDIA)
        - A 시리즈(애플)

    - 번외 : 갤럭시 폰이라고 해서 무조건 Exynos가 들어가는 것은 아니다. 

+ 주기억 장치 : 램(Random Access Memory) 
    - 컴퓨터에서 CPU가 모든 핵심적인 역할을 수행한다면, 주기억 장치는 데이터들과 프로그램을 작업하기 위해 임시 저장하는 역할을 수행함.
    - 데이터들이 어디에 저장되어 있는지 주소값의 형태로 저장하고, 데이터들을 읽고 쓸 수 있다.
    - 전원을 끄면, 주기억장치 내 데이터들이 모두 삭제된다. (휘발성)
    - 주기억장치의 용량이 크면 클수록, 빠른 작업이 가능하다. 컴퓨터그래픽, 3D 그래픽 등을 처리할 때 주기억장치의 용량을 큰 것을 사용하면 더 빨리 작업할 수 있다.
    
    - Dual in-line Memory Module(DIMM) : 키 판에 반도체 메모리칩들이 여러 개 꽂혀져 있는 형태를 말함.

+ 보조 기억장치 
    - 하드 디스크 드라이버(HDD) : 컴퓨터 전원을 꺼도 , 데이터가 그 안에 남아 있음(비휘발성)
        - 주기억 장치에 비해 데이터 접근 속도가 느리다. 
        - 대신 주기억 장치에 비해, 가격이 저렴하고 넓은 공간을 이용할 수 있다.
        - Head와 자기 Disk로 구성되어 있다. 헤드 부분 양쪽에 전류를 흘려 자기화되어 있는 데이터들을 읽고 쓸 수 있다.
    - USB(Universial Serial Bus) : 플래시 메모리(Flash Memory)로 구성되어 있다. (USB에 비해 HDD가 드라이브 접근 시간이 더 빠름.)
    - SSD(Solid State Disk) : 플래시 메모리(Flash Memory)로 구성되어 있음. HDD에 비해 가격이 비싼 대신, 전력 소모와 발열 소음이 적다. 하드 디스크보다 SSD가 충격에도 더 강하며, 데이터 접근 속도 역시 HDD보다 더 빠름. 
    - 외장하드 : 하드 디스크와 구조는 같으나, 플라스틱 커버로 덮여 있다. HDD와 마찬가지로 충격에 약하며, 가격이 저렴. USB나 SSD에 비해서 속도는 더 느리다.

+ 출력 장치 
    - 스피커, 플로터(plotter, 플래카드나 거대한 설계도면을 그릴 때 사용), 모니터 등이 있다.
        - 3D 프린터 : 초기에는 산업 신제품을 만들 때 사용되었지만, 요즘은 많이 보편화되어 많은 사람들이 사용중
        - TV : LCD(Liquid Crystal Display)/LED(Light Emitting Display)/OLED(Organic Light Emitting Display) TV가 있음
        - LCD : 액정에 일정한 전압을 가하면, 결정방향이 줄지어 나타나 뒤쪽에서 빛을 비추었을 때 문자나 그림이 나타남. 출력 속도가 느리고, 시야 각도에 제한이 있음. 
        - LED : LCD와 흡사하지만, 뒤에 발광 다이오드를 사용함. 
        - OLED : 유기발광 다이오드. 자체 형광성 유기화합물에 전류를 흘려, 자체적으로 빛이 발생하게 된다. 
        - UHD(Ultra-High Definition) : 화소(pixel)이 Full-HD 화소보다 4배 높음. 

 
#### 소프트웨어(Software) :
+ 소프트웨어(Software) : 컴퓨터가 일을 할 수 있도록 여러가지 명령어들을 모안 놓은 집합. 이런 것들을 총칭해서 Software라고 이야기함.
+ 시스템 소프트웨어와 응용 소프트웨어로 나누어짐. 
+ 시스템 소프트웨어 
    - 컴퓨터가 작동할 수 있도록 기본적으로 도와주는 기본 소프트웨어. 
    - 운영체제, 유틸리티 프로그램이 기본적으로 시스템 소프트웨어에 들어감.
    - Operating System(OS) : 컴퓨터 시스템의 전반적인 동작을 모두 조정해 컨트롤해 주는 시스템 프로그램
        - 하드웨어와 소프트웨어 중간에 OS가 중간 매개체 역할을 진행한다. 
        - 컴퓨터에 있는 자원을 관리하는 것이 운영체제의 가장 핵심적인 역할이다.
        - 컴퓨터를 부팅하는 BIOS 프로그램도 운영체제의 일부분에 포함된다. (운영체제의 하위에 있는 일부분임)
        - 대표적인 OS : UNIX, MacOS, Android, Windows, Linux, 매킨토시
        - MacOS_몬테레이 : 현재 MacOS의 최신버전. 
        - Utility Program : OS를 도와 컴퓨터 시스템을 원활하게 작동할 수 있도록 유도함. 바이러스 치료, 보안, 디스크 관련 프로그램이 포함됨.
       
+ 응용 소프트웨어 : 특정한 작업에 활용되는 소프트웨어.
    - 예시 : MS Word, Excel, IDE들, MySQL, Oracle, DBMS, SourceTree(깃허브(지금하는 이거) GUI)   


#### 컴퓨터의 기원 
+ 주판 : 기원전 30세기 바빌로니아에서 처음 만들어져, 중국에서 1300년부터 주판이 사용됨. 
+ 파스칼 계산기 : 수학자 파스칼이 톱니바퀴를 이용해 만든 최초의 기계식 계산기.
+ 라이프니츠 계산기 : 파스칼의 계산기를 조금 더 개량해서, 곱셈이나 나눗셈도 계산할 수 있는 계산기. 2진법을 처음으로 만든 사람이기도 하다. 수학시간에 2진법 표시하는 것도, 라이프니츠의 표시를 지금도 사용하고 있다.
+ 분석 엔진 : 영국의 베비지라는 사람이 설계함. 제어/연산/입출력 장치를 포함하고 있고, 입력장치로, 구멍이 뚫린 종이카드를 통해 받아들여 지금 현대 컴퓨터의 모체가 되었다.
+ 천공카드 기계 : 미국의 홀라리스라는 사람이 개발함. 전기와 기계를 사용한 최초의 계산기. 일괄처리 방식(Batch Processing)(복잡한 걔산할 것들 여러 개를 한번에 모아 일괄적으로 처리하는 방식)을 개발함. 현재의 IBM으로 발전함. 
+ 튜링 기계 : 영국의 수학자 엘런 튜링의 이론적인 기계방식. 현대 컴퓨터의 이론적인 모델 제시(그때 당시 구현하지는 못함). 영화 "이미테이션 게임"의 주인공으로 묘사된 적이 있음.

**대체적인 흐름을 파악하면 된다!**

+ 1942년 헝가리 수학자 폰 노이만이 프로그램 내장 방식 개념을 처음으로 제시함.**중요**
    - 프로그램 내장 방식 전에는, 최초의 컴퓨터인 "애니악"이 있던 시절이었다.
    - 프로그램들을 주기억 장치에 두고, 하나씩 불러들여 처리하는 방식이 "폰 노이만"의 프로그램 내장 방식이다.**중요**
+ 에드삭(EDSAC : Electronic Delay Storage Automatic Calculator) : 영국 캠브리지 대학에서, 폰 노이만 프로그램 내장 방식을 최초로 적용한 컴퓨터.
+ ABC(Atanasoff-Berry Computer) : 자동 전자식 디지털 계산기이자 컴퓨터.
+ 하버드 마크-1 : 하버드 대학에서 만들어낸 자동 계산기. 

#### 컴퓨터의 발전
+ 제 1세대 컴퓨터 : 에니악에서 출발해, 에드삭, 에드박 등이 1세대 컴퓨터에 포함된다
    - 에니악 : 진공관과 자기 드럼(보조 기억장치, 원통형의 표면에 마그네틱 물질이 발려져 있음.)을 사용해 만들어짐.
        - 천공 카드를 사용해 입력하고, 컴퓨터가 이해할 수 있는 기계어를 사용했음.
        - 그때그때 프로그램을 실행시킬 때마다 회로를 뜯어서 다시 조립하는 식으로 동작함. 
        - 컴퓨터 내부연산할 때, 10진법을 사용함. 
    - 에드삭 : 폰 노이만의 프로그램 내장 방식을 채택. 쉽게 프로그램 여러 개를 실행시키기 편리해졌다.
    - 에드박 : 에니악을 더 개량,발전시켜 만든 것. 프로그램 내장 방식 채택.
        - 컴퓨터 내부연산 시, 최초로 10진법이 아닌 2진법 연산을 한 최초의 컴퓨터이다.
   
    - 유니박 : 세계 최초의 상업용 컴퓨터. 미국의 통계청에 최초로 설치되었다. (그전 것들은 연구용이기 때문에 상용화되지 못했다)

+ 제 2세대 컴퓨터 : 트랜지스터가 진공관을 대체해, 컴퓨터의 부피가 작아지고 성능은 우수해졌으며, 가격은 더 저렴해짐. 
    - 주기억 장치로는 "자기 코어"를 사용함. 보조기억장치로는 자기 디스크 형태의 보조기억 장치를 사용함.  
    - 1956년 IBM 704 라는 컴퓨터를 실행시키기 위해서 운영체제(OS)가 최초로 개발되었다.
    - 최초의 운영체제 : GM-NAA(제너럴 모터스에서 개발됨)
    - 컴퓨터 안에서 작동시킬 수 있는 고급 수준의 프로그래밍 언어가 개발되었음(FORTRAN, 코볼 등)
    - 소프트웨어가 본격적으로 발전하기 시작한 세대이다.

+ 제 3세대 컴퓨터 : 트랜지스터에서 집적 회로로 넘어간 시기. 트랜지스터, 다이오드, 저항 같은 소자들을 한 곳에 반도체로 넣어 집적 회로로 만듬. 
    - IC(Intergrated Circuit)을 사용하여 컴퓨터 회로를 구성함. 
    - 아직 개인 컴퓨터가 나오기 전이었다. 
    - 한 군데에서 대형 컴퓨터를 돌리고, 여러 사람이 이 컴퓨터에 연결해 터미널을 가지고 사용했음(메인 프레임 시스템)
    - 그래서 여러 사람이 돌리는 여러 프로그램이 Round Robin 방식으로 다중프로그래밍을 통해서 프로그램을 돌렸다.(시분할 운영체제 Multics)
    - PL/1 이라는 프로그래밍 언어를 통해 Multics 운영체제가 개발되었고, 이런 멀티 프레임 시스템에서 여러 사람들이 자기가 사용하고 싶은 프로그램을 프레임에서 사용할수 있도록 운영체제가 개발됨.
    - 이러한 운영체제 하에서, basic(베이직) 이라는 언어가 개발됨. 

+ 제 4세대 컴퓨터 : 집적회로가 고밀도로 개발됨. 슈퍼컴퓨터의 탄생.
    - LSI (Large Scaled Intergrated) 
    - Intel 사의 마이크로프로세서 8080이라는 것이 개발되어, 최초의 개인용 PC가 탄생.
        - 모니터, 키보드가 없는 컴퓨터였음. LED 불이 깜빡이는 것으로 출력을 대신함. 
        - 입력 장치는 스위치였고, 올리고 내리면서 입력을 주게 되었음.
        - 베이직 언어를 더 개선해서 "알테이어 베이직" 언어를 개발함(빌 게이츠 & 폴 앨런)
        - 해당 언어를 판매하기 위해 MS사가 설립됨. 

    - 스티브 잡스와 스티브 뮤즈니아가 협력해 애플 사를 창업함.(애플 원 컴퓨터의 탄생, 요즘의 컴퓨터와 흡사)
    - 1982년, IBM은 MS-DOS를 탑재한 IBM PC/XT를 발표함.
    - 애플은 최초의 GUI 형태의 운영체제를 사용한 Lisa 컴퓨터를 발표함.(최초의 GUI 운영체제는 MS보다 애플이 더 빨랐다)
    - 이후 웹 서비스 개발/웹 브라우저 사용됨
    - 미국의 Cray 사에서 슈퍼컴퓨터를 만듬.

+ 제 5세테 컴퓨터 : 앞으로 다가올 컴퓨터. 컴퓨터가 인간처럼 생각하고, 인간과 대화하며, 자연어로 명령을 내리면 창의적으로 작업을 수행하는 단계의 컴퓨터.

    - 양자컴퓨터를 사용해 연구원들과 과학자들이 연구중. 양자물리학의 원리를 사용해 "큐 비트"로 데이터를 처리하고 있음. 
        - 큐 비트 : 0과 1 뿐 아니라, 0과 1이 중첩되어 있는 상태가 새로 나오게 되면서 이러한 비트처리를 "큐 비트"라고 한다. 
        - 0과 1뿐아니라, 새로운 비트가 하나 더 들어갔기 때문에 처리속도가 그만큼 되게 빨라진다고 한다.
        - 양자 컴퓨터는 슈퍼 컴퓨터가 150년동안 계산할 양을 단 4분만에 처리할 수 있다고 한다. 
        - 양자 컴퓨터는 캐나다에서 최초로 개발되었다고 한다.
    
    - MS 사에서도 양자컴퓨터에서 실행시킬 수 있는 프로그래밍 언어를 개발했고, 구글에서도 양자컴퓨터를 개발중.
    - 대량의 데이터를 받아들여 처리할 수 있기 때문에, 사물인터넷이나 빅데이터, 인공지능 등의 분야에서 사용될 수 있다.


#### 컴퓨터의 종류
+ 슈퍼컴퓨터 : 주로 과학기술 분야에서 복잡한 계산 전용으로 사용하는 컴퓨터. 
    - 기상청이나, 은행, 대기업 연구소, 영화 특수효과 삽입 등의 분야에 사용될 수 있다. 
    - 의&약학 분야에서의 유전자 염기서열 분석에도 사용될 수 있다. 
    - 미국의 Cray 사가 슈퍼컴퓨터로 유명하며, IBM에서도 슈퍼컴퓨터를 만들어 낸다. 중국에서도 만들어냄.
    - 슈퍼컴퓨터의 계산능력을 나타내는 지표의 단위는 FLOPS(FLoating point Opeations Per Second)라는 단위를 사용함.

+ 메인 프레임 : 한 컴퓨터에서 작업을 모두 처리하고, 단말기로 연결해서 여러 사람들이 동시에 사용할 수 있는 컴퓨터.
    - 대기업, 은행, 대학교 연구소에서 각각 단말기 터널을 이용해 여러 사람의 사용자가 함께 쓸 수 있는 대형 컴퓨터이다.

+ 마이크로 컴퓨터 : 요즘 사용하는 개인용 컴퓨터. 
    - IBM 호환 컴퓨터, 애플의 Mac 컴퓨터.

+ 워크스테이션 : 1980~1990년대에, 일반 PC의 성능이 대단히 떨어지던 시대에 사용되던 컴퓨터. 여러 가지 고도의 수치처리능력이 필요한 작업, 그래픽 작업, 기계보급 설꼐, 3차원 모델링, 애니메이션 작업 등의 그래픽 관련 작업을 할 때 사용되던 컴퓨터이다.
    - 요즘 나오는 컴퓨터들은 옛날 워크스테이션 성능보다도 더 월등하기 때문에, 일반 PC로도 컴퓨터그래픽 관련 일들을 처리할 수 있다. 

+ 미니 컴퓨터 : 성능과 크기 면에서 메인 프레임과 워크스테이션, 또는 마이크로 컴퓨터 사이 정도의 컴퓨터. 메인프레임 컴퓨터보다는 저렴했다. 

+ 휴대형 컴퓨터 : 스마트폰, 태블릿PC, 노트북, PDA
    - 노트북 : Laptop, Palmtop(손바닥 위에 들어간다고 하여 지어진 이름)
    - PDA : 우체국, 택배배달 기사님들이 들고 다니는 것들.

+ 개발 실험용 컴퓨터 : 아두이노, 라즈베리 파이
    - 간단하고 저렴한 가격으로, 간단한 기능을 하는 컴퓨터 같은 것들을 작동시킬 수 있다.
    - 운영체제는 리눅스 계열이다. 
    - 아두이노는 교육용으로 사용될 수 있는 초소형 컴퓨터이다.
